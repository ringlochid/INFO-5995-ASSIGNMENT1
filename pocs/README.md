POCs
This folder contains code-level proof-of-concept helpers.

- `SessionTokenPOC.java` (in `findings/`) demonstrates `Random` token prediction logic.
- `CredentialsPOC.java` (in `findings/`) demonstrates parsing plaintext credentials.

## Emulator Frontends

- `pocs/session-token-emulator/` contains a standalone HTML/CSS/JS emulator plus a copy of `SessionTokenPOC.java`.
- `pocs/credentials-emulator/` contains a standalone HTML/CSS/JS emulator plus a copy of `CredentialsPOC.java`.

Open the `index.html` file in each folder to run the emulator locally.
